# Project Rules

## Code Formatting

- **Indentation**: Use **4 spaces** for all files (HTML, CSS, JS, JSON, Python)
- Never use tabs or 2-space indentation
- This rule applies to all code in this project

## Project Overview

Interactive map for displaying advertising points using Yandex Maps API 3.0.

## Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript (ES6 modules)
- Yandex Maps API 3.0 with clustering
- No frameworks

## Data Files

- `data/map_prices.csv` - Source data with points (editable by users)
- `data/mappings/region-mapping.json` - Region normalization
- `data/mappings/brand-mapping.json` - Brand grouping

## Language

- Code: English (variable names, comments)
- UI: Russian
- Documentation: Russian or English
